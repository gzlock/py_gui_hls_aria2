# HLS视频录制GUI工具

### 特性
- 支持可变分辨率的VOD和Live的HLS m3u8资源
- 默认会选取最高分辨率的分支进行录制
- HLS的ts视频碎片使用集成的Aria2进行下载，安全可靠
- 支持Windows和macOS系统

### 一体化
- 集成Aria2c，下载HLS的视频碎片
- 集成FFmpeg，将HLS的视频碎片合并为完整视频

### 工作流程
1. 点击开始录制按钮
1. 读取一次m3u8源判断m3u8源和代理(如果有设置)的可用性
1. 不断读取m3u8源，分析出还没有下载的碎片视频链接数组
1. 将没有下载的碎片视频链接数组交给Aria2下载
1. 点击停止录制按钮
1. 使用合并视频功能将视频碎片合并为完全的视频文件
